# 账务系统 - 代币预充值与消费体系

## Goal
实现代币预充值系统，包含充值、划拨、消费记录查询功能，支持多角色权限控制。

## Context
本系统采用代币预充值模式：
- 用户提前充值代币，消费时扣除代币
- Admin 可以给任何人划拨代币（包括自己）
- VIP（专业组）可以给自己和下属充值
- User（普通用户）只能充值，不能划拨

## Decisions Made
| Decision | Rationale |
|----------|-----------|
| 使用代币余额在用户端展示 | 遵循用户需求，预充值模式 |
| 按角色划拨：Admin可划拨给任何人，VIP只能划拨给下属 | 权限层级设计，参考用户需求 |
| 消费记录包含：时间、操作类型、消费数量、余额 | 完整账务追溯需求 |
